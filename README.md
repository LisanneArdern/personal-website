# Personal Website (React + Vite)

This project has been modernized from a static HTML/CSS site to a React app
powered by Vite.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview the production build:

```bash
npm run preview
```
